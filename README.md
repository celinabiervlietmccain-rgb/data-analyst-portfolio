# Sales Performance Analysis — Portfolio Project

This project analyzes retail sales data using Excel to identify trends in revenue, regional performance, product categories, and top customers.

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Calculated Fields  
- Data Visualization  

## Key Insights
- Total revenue analyzed: **$2.26M**
- The West region generated the highest sales
- Sales peaked during **Q4 (Oct–Dec)**
- Technology was the highest-performing category
- A small group of customers contributed a significant portion of revenue
- Monthly sales trends show seasonal patterns

## Project Purpose
This project demonstrates foundational data analysis skills including:
- Data cleaning  
- Aggregation  
- Visualization  
- Communicating business insights
